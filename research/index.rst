Toolset Investigation
#####################

.. toctree::
   :hidden:

   tools
   docutils/index
   sphinx/index

When you need to gain deep knowledge about a toolset, the general description provided in the toolset's
official documents are often not sufficient. In this case, you might need to trace the execution process
to find answers to your specific questions.

The approach described here might be generally helpful in a code debugging process as well.
