Tutorials¶

This is a set of guides that will help you step by step to gain experience in solving the main practical problems of documentation.

You can learn more from the following resources:

  • Documenting Your Project in Sphinx, PyCon 2013, Brandon Rhodes, YouTube recording (2h 21m)

  • Handout Materials and Sample Project by Brandon Rhodes

  • Practical Sphinx PyCon 2018, Carol Willing, YouTube recording (27m)

  • Sphinx - How to generate documentation from python doc strings, 2022, by Joel, YouTube recording (27m)

  • Your code should document itself!, 2018, Bryson Tyrrell, YouTube recording (41m)

  • Building Docs like Code: Continuous Integration for Documentation, 2019, Mason Egger, YouTube recording (29m)

  • Use Gitlab to Deliver “Docs-as-Code” Technical Documentation, 2021, Commit Virtual 2021, Alec Clews, YouTube recording (25m)

  • Sphinx for Python Documentation Tutorial, 2021, Melissa Weber, YouTube recording (1h 13m)
